Projects

Sistema de Gestión de Biblioteca:

Desarrollo de un sistema completo de gestión de biblioteca que permite a los usuarios buscar, prestar y devolver libros. El sistema incluye funcionalidades de administración para agregar nuevos libros, actualizar inventarios y gestionar préstamos. Tecnologías utilizadas: HTML, CSS, JavaScript, PHP y MySQL.

Plataforma de E-learning Interactiva:

Creación de una plataforma educativa en línea que ofrece cursos interactivos en una variedad de temas, incluyendo programación, diseño gráfico y desarrollo personal. La plataforma proporciona a los usuarios un entorno de aprendizaje dinámico con videos, cuestionarios y proyectos prácticos. Tecnologías utilizadas: React.js, Node.js, MongoDB y Redux.

Aplicación Móvil de Entrega de Comida:

Desarrollo de una aplicación móvil de entrega de comida que permite a los usuarios realizar pedidos en línea desde una variedad de restaurantes locales. La aplicación ofrece una interfaz intuitiva para buscar restaurantes, explorar menús y realizar pagos seguros. Tecnologías utilizadas: Flutter, Firebase y Google Maps API.
